表的约束

常用的约束：

  • 主键约束(Primary Key Constraint)：要求主键列数据唯一，并且不允许为空
  • 唯一约束(Unique Constraint)：要求该列数据唯一，允许为空，但只能出现一个空值
  • 检查约束(Check Constraint)：某列取值范围限制、格式限制等
  • 默认约束(Default Constraint)：某列的默认值
  • 外键约束（Foreign Key Constraint)：用于两表间建立关系，需要指定引用主表的哪列

约束可以在创建表的时候直接添加(推荐)

创建表后使用 sql 语句添加

约束名的取名规则(推荐)

  • 主键(Primary Key)约束：如 PK stuNo
  • 唯一(Unique Key)约束：如 UQ stulD
  • 默认(Default Key)约束：如 DF stuAddress
  • 检查(Check Key)约束：如 CK stuBornDate
  • 外键(Foreign Key)约束：如 FK_stuNo
