标识符命名规则

  • 只能由数字、字母、下划线_、美元符号$、中文字符组成

  • 大小写敏感
  • 不能是关键字

潜规则：

  1. 见名知意
  2. 一般以小写开头、最好使用驼峰命名法

数组

java的数组是放在堆里的

有两种声明方式

  • int[] a = {1,2,3};
  • int a[] = {1,2,3};

不赋初值的声明形式

  • int[] a = new int[10]; 数组长度为10
  • int a[] = new int[10];

不赋初值情况下每个元素的值都为默认值

  • int、double的默认值为0
  • boolean的默认值为false
  • String的默认值为“null”

错误提示出现说明是数组长度不够

获取数组长度：数组名.length

方法

程序 = 数据 + 命令 + 逻辑

方法(method)也可以叫函数(function)

方法就是组成程序的可以“重复使用”的积木，方法把复杂的N行代码封装起来，让使用者可以通过方法名调用（使用）。

方法由方法名、参数、返回值组成

方法同样支持重载（同名方法但参数不同）

类

基本语法

  • private：只能类内部访问
  • public：其他类也能访问
  • 类的定义由成员变量和方法组成

语法：

  • 如果类中的方法为static类型,可以直接调用

  • 对象的赋值

成员初始化

  • 所有类型的局部变量必须初始化，而成员变量声明是可以不初始化
  • 基本类型的默认初始值为0、false
  • String等非基本类型初始化为null

NULL

  • 当一个变量指向NULL的时候表示：变量不指向任何对象
  • java的基本数据类型(byte、short、int、long、float、double、char、boolean)不可被赋值为null
  • 一般遇到报错NullPointerException为所用对象赋值为null，没有指向任何一个对象

JavaBean

  • 按照JavaBean的语法规范来编写，能够更简单的和其他框架一起使用

规则：

  1. 必须是public类
  2. 成员变量用private，通过public的setter、getter方法访问。

继承

  • 继承用来简化类的声明
  • Java中的类都直接或间接继承自object类

基本语法

  • 父类类型的变量可以指向子类类型的对象
  • 显示转换，程序员确认p1指向子类对象

  • 如果路径中复制出的路径存在\则需要改成/
